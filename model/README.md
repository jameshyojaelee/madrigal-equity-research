# rNPV model

This folder houses the risk-adjusted NPV model for Madrigal (MDGL), focused on Rezdiffra in MASH. The structure is purely educational and transparent, and nothing herein constitutes investment advice.

## Files
- `create_rnpv_template.py` – script that generates the initial Excel template.
- `madrigal_rnpv_model_template.xlsx` – generated template output (not committed if large).
